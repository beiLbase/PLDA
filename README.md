PLDA is a method that combines prompt learning and data augmentation for few-shot relation extraction.
PLDA method is based on the followings papers： “FewRel: A Large-Scale Supervised Few-Sample Relation Classification Dataset with State-of-the-Art Evaluation Results” and “PTR: Prompt Tuning with Rules for Text Classification”

The orginal code and data: https://github.com/thunlp/FewRel
The link of the orginal code：https://github.com/thunlp/PTR
