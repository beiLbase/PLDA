PLDA is a method that combines prompt learning and data augmentation for few-shot relation extraction.
PLDA method is based on the followings papers： “FewRel: A Large-Scale Supervised Few-Sample Relation Classification Dataset with State-of-the-Art Evaluation Results” and “KnowPrompt: Knowledge-aware Prompt-tuning with Synergistic Optimization for Relation Extraction”

The orginal code and data: https://github.com/thunlp/FewRel

The link of the orginal code：https://github.com/zjunlp/KnowPrompt

The doi of the orginal paper: https://doi.org/10.1145/3485447.3511998
